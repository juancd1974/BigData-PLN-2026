NormaSearch — Motor de búsqueda semántica para normatividad colombiana

Descripción del Proyecto

NormaSearch aplica Procesamiento de Lenguaje Natural para analizar, indexar y recuperar normatividad colombiana. Permite búsquedas semánticas, extracción de entidades nombradas y generación de resúmenes automáticos de actos administrativos.

Tecnologías utilizadas

  • Python — backend principal
  • Flask — framework web
  • Elasticsearch — búsqueda e indexación
  • spaCy — entidades y análisis POS
  • Transformers — resúmenes con mT5
  • gensim — Word2Vec (CBOW y Skip-gram)
  • scikit-learn — TF-IDF y clasificación
  • MongoDB — gestión de usuarios
  • Bootstrap — interfaz responsiva
